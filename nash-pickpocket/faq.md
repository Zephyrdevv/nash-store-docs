# FAQ

**Q: The pickpocket animation is not playing.**
A: Ensure ox_lib is started and the animation dictionary is loaded.

**Q: How do I change what items are available to steal?**
A: Edit the `Config.LootTable` entries in config.lua.

**Q: Can I increase success rates?**
A: Adjust `Config.SuccessChance` in config.lua or add job/skill modifiers.

**Q: Can police be alerted on failed attempts?**
A: Yes, set `Config.CopAlert = true` in config.lua.

**Q: Is there a cooldown between pickpocket attempts?**
A: Yes, set `Config.Cooldown` to the desired seconds between attempts.
