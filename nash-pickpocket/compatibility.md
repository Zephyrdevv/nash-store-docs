# Compatibility

## Frameworks

| Framework | Supported |
|-----------|-----------|
| ESX (es_extended) | Yes |
| QBCore | No |
| QBx | No |

## Required Scripts

| Script | Purpose |
|--------|---------|
| es_extended | Core framework |
| ox_lib | Utility library for UI and callbacks |
| ox_inventory | Inventory management |
| ox_target | NPC targeting |

## Optional Scripts

| Script | Purpose |
|--------|---------|
| p_mdt | Police MDT — creates dispatch alerts |
