# ducky_multicharacter

ducky_multicharacter is a multi-character selection system for FiveM. Players can create, manage, and switch between multiple characters on a single account with a polished NUI selection screen.

## Features

- Multiple character slots per account
- Character creation with name, date of birth, and appearance
- Character deletion with confirmation
- Smooth camera transitions on the selection screen
- ESX and QBCore compatible
- Spawn position per character

## Requirements

- ox_lib
- ESX or QBCore
- oxmysql
