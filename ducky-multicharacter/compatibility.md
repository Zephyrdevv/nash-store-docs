# Compatibility

| Framework | Compatible |
|-----------|------------|
| ESX | Yes |
| QBCore | Yes |
| QBx | Yes |
| Standalone | No |

## Dependencies

| Resource | Required |
|----------|----------|
| ox_lib | Yes |
| oxmysql | Yes |
| ESX or QBCore | Yes |

## Known Conflicts

- `esx_multicharacter` — disable if using ducky_multicharacter
- `qb-multicharacter` — disable if using ducky_multicharacter
