# FAQ

**Q: Characters are not saving.**
A: Ensure oxmysql is running and the SQL file was imported correctly.

**Q: Can I increase the number of character slots?**
A: Yes, change `Config.MaxCharacters` in config.lua.

**Q: The selection screen camera is in the wrong position.**
A: Adjust `Config.CameraCoords` in config.lua to match your desired camera angle.

**Q: How do I set different spawn points per character?**
A: Spawn points can be set per-character through the creation menu or configured in config.lua.

**Q: Can I disable character deletion?**
A: Set `Config.EnableDeletion = false` in config.lua.
