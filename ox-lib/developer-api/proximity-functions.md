# Proximity Functions

## lib.points.new(data)
Creates a new proximity point.

| Parameter | Type | Description |
|-----------|------|-------------|
| data.coords | vector3 | World coordinates |
| data.distance | number | Detection radius |
| data.onEnter | function | Called when player enters |
| data.onExit | function | Called when player exits |

## point:remove()
Removes the proximity point.
