# Compatibility

| Framework | Compatible |
|-----------|------------|
| ESX | Yes |
| QBCore | Yes |
| QBx | Yes |
| Standalone | Yes |

ox_lib is framework-agnostic. Nash Edition maintains full compatibility with all standard ox_lib integrations while adding the Nash UI theme.

## Supported Game Builds

- FiveM (latest recommended)
- GTA V build 2060+
