# FAQ

**Q: Can I use the standard ox_lib instead of Nash Edition?**
A: Nash Store scripts require Nash Edition for correct UI styling. Standard ox_lib may work but the visual theme will differ.

**Q: Does Nash Edition receive upstream ox_lib updates?**
A: Nash Edition is periodically rebased on the latest ox_lib stable releases. Check the Nash Store updates channel for version notes.

**Q: Can I change the color theme?**
A: The primary color can be overridden via the `ox:primaryColor` ConVar in server.cfg.

**Q: Is Nash Edition open source?**
A: Nash Edition is a commercial product. The underlying ox_lib code is open source (LGPL). The Nash NUI customizations are proprietary.
