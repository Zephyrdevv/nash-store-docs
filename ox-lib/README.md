# ox_lib — Nash Edition

ox_lib (Nash Edition) is a customized build of the Overextended ox_lib utility library for FiveM. Nash Edition includes a custom color palette, styled NUI components, and additional helper functions tailored for Nash Store scripts.

## Features

- Custom Nash color theme baked into the NUI build
- Notification system (lib.notify)
- Progress bars (lib.progressBar)
- Input dialogs (lib.inputDialog)
- Alert/confirm dialogs (lib.alertDialog)
- Proximity point system (lib.points)
- Zone management (lib.zones)
- Callback system (lib.callback)
- Command registration helpers

## Requirements

- FiveM server
- Compatible with ESX, QBCore, QBx, and standalone resources
