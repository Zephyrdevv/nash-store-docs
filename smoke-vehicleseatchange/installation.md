# Installation

## Requirements

- FiveM server
- ox_lib (optional)

## Steps

1. Place `smoke_vehicleseatchange` in your resources directory.
2. Add to `server.cfg`:

```
ensure smoke_vehicleseatchange
```

3. Optionally configure the keybind in `config.lua`.
4. Restart your server.

## Keybind

Default keybind: `G` (configurable in config.lua)
