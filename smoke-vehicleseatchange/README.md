# smoke_vehicleseatchange

smoke_vehicleseatchange allows players to smoothly switch seats inside vehicles without exiting. This is a lightweight, standalone utility script.

## Features

- Switch between driver, passenger, and rear seats
- Configurable keybind
- Animation played on seat switch
- Works with all vehicle types
- Standalone (no framework required)

## Requirements

- ox_lib (optional, for notifications)
