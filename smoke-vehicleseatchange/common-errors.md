# Common Errors

### Keybind not working
**Cause:** Conflicting keybind or resource not started.
**Solution:** Change `Config.Keybind` in config.lua and ensure the resource is ensured in server.cfg.

### Seat switch not animating
**Cause:** Animation files not loaded.
**Solution:** Ensure the resource is not corrupted. Re-download and replace.

### Script not loading
**Cause:** fxmanifest.lua syntax error.
**Solution:** Check the fxmanifest.lua for syntax errors and ensure the resource folder name matches.
