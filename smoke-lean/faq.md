# FAQ

**Q: The item is not being removed on use.**
A: Set `Config.UseItem = true` and ensure ox_inventory or your framework inventory is running.

**Q: Can I add custom animations?**
A: Yes, change `Config.AnimDict` and `Config.AnimName` to any valid GTA V animation.

**Q: How do I change the screen effect?**
A: Set `Config.ScreenEffect` to a valid GTA V screen effect name.

**Q: Can I add multiple drug types?**
A: Yes, use the `Config.Substances` table to define multiple items with individual effects.
