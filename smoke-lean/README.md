# smoke_lean

smoke_lean adds immersive drug use animations and effects to FiveM servers. Players can consume lean (and other configurable substances) with realistic animations, screen effects, and optional item consumption via ox_inventory.

## Features

- Configurable drug item names
- Animation and screen effect on use
- Duration and intensity settings
- Optional item removal on use
- ESX, QBCore, and QBx compatible
- Notify on use/overdose via ox_lib

## Requirements

- ox_lib
- ESX or QBCore
- ox_inventory (optional, for item-based use)
