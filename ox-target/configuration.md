# Configuration

## config.lua

| Option | Default | Description |
|--------|---------|-------------|
| DrawDistance | 7.5 | Max distance to show target |
| BoxZoneDebug | false | Draw debug outlines for zones |
| RaycastEntities | true | Enable entity raycasting |
| TargetBone | false | Target specific bones |

## Nash Theme

The interaction menu uses the Nash color palette by default. Colors are set via ox_lib ConVars:

- `ox:primaryColor` — accent color for selected options
- `ox:errorColor` — used for delete/remove options
