# ox_target — Nash Edition

ox_target is a target/interaction system for FiveM. The Nash Edition ships with a custom styled interaction menu and expanded helper exports.

## Features

- Entity targeting (peds, vehicles, objects)
- Zone-based targeting
- Global options
- Custom draw distance and style
- Fully compatible with ESX, QBCore, and QBx

## Included Files

- ox_target resource (NUI customized)
- Nash preset configuration
- Example usage scripts
