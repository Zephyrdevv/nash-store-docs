# Entity Exports

## addLocalEntity(entity, options)
Adds target options to a specific entity handle.

| Parameter | Type | Description |
|-----------|------|-------------|
| entity | number | Entity handle |
| options | table | Array of option tables |

## removeLocalEntity(entity, optionNames)
Removes specific options from a local entity.

## addSphereZone(data)
Adds a sphere zone with target options.

## removeZone(id)
Removes a zone by its id.
