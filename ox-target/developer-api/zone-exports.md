# Zone Exports

## addBoxZone(data)
Creates a box-shaped interaction zone.

| Parameter | Type | Description |
|-----------|------|-------------|
| data.coords | vector3 | Center of the box |
| data.size | vector3 | Box dimensions |
| data.rotation | number | Heading rotation |
| data.options | table | Target options |

## addPolyZone(data)
Creates a polygon interaction zone.

## addSphereZone(data)
Creates a sphere interaction zone.
