# FAQ

**Q: The barber menu does not open.**
A: Ensure ox_target is running and the barber location coords are correct.

**Q: Changes are not saving after purchase.**
A: Verify your database connection and that the SQL file was imported.

**Q: Camera is not showing during customization.**
A: Set `Config.CameraEnabled = true` in config.lua.

**Q: Can I add multiple barber locations?**
A: Yes, add additional entries to `Config.Locations` in config.lua.

**Q: Does this work with custom ped models?**
A: Customization is limited to standard GTA V component data. Custom ped support may vary.
