# Server Callbacks

## dukcy_barbershop:getAppearance
Returns the saved appearance for the player.

**Returns:** table with saved hair, beard, eyebrows, makeup component data.

## dukcy_barbershop:canAfford
Checks if the player can afford a given category.

| Parameter | Type | Description |
|-----------|------|-------------|
| category | string | Category to check |

**Returns:** boolean
