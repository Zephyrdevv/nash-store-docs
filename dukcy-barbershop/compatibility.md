# Compatibility

| Framework | Compatible |
|-----------|------------|
| ESX | Yes |
| QBCore | Yes |
| QBx | Yes |
| Standalone | Partial |

## Dependencies

| Resource | Required |
|----------|----------|
| ox_lib | Yes |
| ox_target | Yes |
| ox_inventory | No (optional for item-based payment) |

## Known Conflicts

- `qb-barbershop` — disable if using dukcy_barbershop
- `esx_barbershop` — disable if using dukcy_barbershop
