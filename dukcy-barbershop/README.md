# dukcy_barbershop

dukcy_barbershop is a fully featured barbershop script for FiveM. Players can customize their hair, beard, eyebrows, and makeup through an immersive NUI menu at barber locations.

## Features

- Full hair, beard, eyebrows, and makeup customization
- Multiple barber locations configurable via config
- ox_lib progress bar and notifications integrated
- ESX and QBCore compatible
- Camera preview while customizing
- Purchase confirmation dialog
- Ped outfit restore on cancel

## Requirements

- ox_lib
- ox_target
- ESX or QBCore
