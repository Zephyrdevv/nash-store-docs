# Developer API

BagPick relies on the following external exports:

| Export | Resource | Usage |
|--------|----------|-------|
| `exports['es_extended']:getSharedObject()` | es\_extended | Retrieves the ESX shared object |
| `exports.ox_target:addGlobalPed()` | ox\_target | Adds targeting options to all peds |
| `exports["vms_notify"]:Notification()` | vms\_notify | Triggers in-game notifications |
