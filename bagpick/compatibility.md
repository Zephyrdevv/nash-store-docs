# Compatibility

## Frameworks

| Framework | Supported |
|-----------|-----------|
| ESX (es\_extended) | Yes |
| QBCore | No |
| QBx | No |

## Required Scripts

| Script | Purpose |
|--------|---------|
| es\_extended | Core framework |
| mysql-async | Database driver |
| ox\_target | NPC targeting interaction |
| vms\_notify | In-game notifications |

## Optional Scripts

| Script | Purpose |
|--------|---------|
| qs-dispatch | Police dispatch alerts (alternative dispatch system) |
