# First Launch

## Console checks

On server start, you should see these lines in the console:

```
[Nash Banking] Framework detected: esx (obj: ESX)
```

_Full list of expected boot messages — filled in Pass 2._

## In-game checks

1. Walk to a Fleeca ATM → press `E` → the ATM UI should open with a card insertion animation.
2. Walk to a bank agency ped → press `E` → the full desktop banking UI should open.
3. Open LB Phone → the two Nash apps should be listed.

## If something doesn't work

Head over to [Common Errors](../common-errors.md).
