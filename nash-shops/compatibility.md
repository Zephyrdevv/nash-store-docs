# Compatibility

| Framework | Compatible |
|-----------|------------|
| ESX | Yes |
| QBCore | Yes |
| QBx | Yes |
| Standalone | Partial |

## Dependencies

| Resource | Required |
|----------|----------|
| ox_lib | Yes |
| ox_target | Yes |
| ox_inventory | Yes |

## Known Conflicts

- `esx_shops` — disable if using nash_shops
- `qb-shops` — disable if using nash_shops
