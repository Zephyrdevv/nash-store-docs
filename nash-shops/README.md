# nash_shops

nash_shops is a configurable general store/shop system for FiveM. It supports multiple shop types, custom item lists, and integrates with ox_inventory and ox_target.

## Features

- Multiple shop types (24/7, liquor store, hardware, etc.)
- Configurable item lists per shop
- ox_inventory integration for item transactions
- ox_target interaction at shop locations
- ESX and QBCore compatible
- Optional robbery mechanic

## Requirements

- ox_lib
- ox_target
- ox_inventory
- ESX or QBCore
