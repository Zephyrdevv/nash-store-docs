# nash_clothesshop

nash_clothesshop is a feature-rich clothing shop script for FiveM. Players can browse, try on, and purchase clothing items through an immersive NUI interface at configurable shop locations.

## Features

- Full GTA V component and prop customization
- Try-before-you-buy preview
- Outfit saving and loading
- Multiple shop locations
- ESX and QBCore compatible
- ox_lib UI integration
- ox_target interaction

## Requirements

- ox_lib
- ox_target
- ESX or QBCore
- oxmysql (for outfit saving)
