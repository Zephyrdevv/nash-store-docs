# FAQ

**Q: The clothing menu does not open.**
A: Ensure ox_target is running and the shop location coords match in-game positions.

**Q: Outfits are not saving.**
A: Ensure oxmysql is running and the SQL file was imported.

**Q: Can I add more outfit slots?**
A: Yes, change `Config.MaxOutfits` in config.lua.

**Q: How do I restrict certain clothing items?**
A: Edit the component/prop tables in `config.lua` to remove unwanted entries.

**Q: Does this support MP freemode clothing?**
A: Yes, all standard GTA V freemode male/female components and props are supported.
