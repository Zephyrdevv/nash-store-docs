# Compatibility

| Framework | Compatible |
|-----------|------------|
| ESX | Yes |
| QBCore | Yes |
| QBx | Yes |
| Standalone | Partial |

## Dependencies

| Resource | Required |
|----------|----------|
| ox_lib | Yes |
| ox_target | Yes |
| oxmysql | Yes (for outfit saving) |

## Known Conflicts

- `esx_clotheshop` — disable if using nash_clothesshop
- `qb-clothing` — disable if using nash_clothesshop
